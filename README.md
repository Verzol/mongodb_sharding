# Mô phỏng MongoDB: Replication và Sharding

Repo này chứa 2 mô phỏng cho các kiến trúc MongoDB, dựa trên bài viết của GeeksforGeeks.
Cả 2 mô phỏng đều sử dụng Docker và Docker Compose.

## 1. Mô phỏng Replication (Replica Set)

* **Mục đích:** Mô phỏng tính năng High Availability (Sẵn sàng cao) và Fault Tolerance (Chịu lỗi).
* **Chi tiết:** [Xem hướng dẫn trong thư mục replication-demo](./replication-demo/)

## 2. Mô phỏng Sharding (Sharded Cluster)

* **Mục đích:** Mô phỏng tính năng Horizontal Scaling (Mở rộng ngang) để xử lý lượng dữ liệu lớn.
* **Chi tiết:** [Xem hướng dẫn trong thư mục sharding-demo](./sharding-demo/)
